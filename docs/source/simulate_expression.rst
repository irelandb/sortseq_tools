==========================================
``simulate_expression``
==========================================

.. contents::

Overview
-------------
``simulate_expression`` is a program within the sortseq_tools package which simulates
an MPRA experiment. 

After you install `sortseq_tools`_, this program will be available to run at the command line. 

Command-line usage
---------------------

   :func: parser
   :prog: sortseq
